ВРЕМЕННЫЕ ИКОНКИ ДЛЯ ТЕСТИРОВАНИЯ
Для работы расширения нужны файлы:
1. icon16.png (16x16 пикселей)
2. icon48.png (48x48 пикселей)
3. icon128.png (128x128 пикселей)
Вы можете:
1. Создать простые иконки в Paint (16x16, 48x48, 128x128)
2. Скачать с https://icons8.com/icons (бесплатно)
3. Использовать любые простые PNG файлы
Для тестирования можно использовать любые PNG файлы с этими именами.
